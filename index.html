2. Define HTML boilerplate code

ANSWER

HTML boilerplate code are sections of code that are repeated in multiple places with little to no variation. A boilerplate in HTML is a template you will add at the start of your project. Also , An HTML Boilerplate is a pre-prepared library of code that can be copied and is meant to be used as a starting point to start building web projects. It is sometimes referred to as an HTML starter template, HTML framework, HTML skeleton, or HTML foundation.

3.Differentiate between the Ordered and Unordered lists.

ANSWER A An unordered list (ul) is used to create a list of items in no particular order i.e. the order of items is not relevant. By default, the items in this list will be marked with bullets. Whereas, an ordered list (ol) is used to create a list of items in a specific order.
